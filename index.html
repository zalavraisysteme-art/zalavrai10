📋 Guide de connexion


  
  

    

      
1

      

        
Créer le projet Supabase

        
supabase.com → New Project (gratuit)

      

    

    

      → supabase.com → Se connecter → New Project

      → Nommer : zalavrai · Région : Europe West

      → Attendre ~2 minutes que le projet démarre
    

  


  
  

    

      
2

      

        
Créer les tables SQL

        
Copier-coller schema.sql → Run

      

    

    

      → Supabase → SQL Editor → New Query

      → Coller tout le contenu du fichier schema.sql

      → Cliquer ▶ Run → 6 tables créées

      → Storage → New bucket → zalavrai-photos · Public ✅
    

  


  
  

    

      
3

      

        
Copier les clés API

        
Settings → API → Project URL + anon key

      

    

    

      → Supabase → Settings → API

      → Copier Project URL → coller dans le champ URL ci-dessus

      → Copier anon public key → coller dans le champ Clé

      → Cliquer ☁️ Connecter Supabase
